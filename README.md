# threaten-push
## 项目说明

漏洞频发的时代，及时获取到漏洞信息是非常有必要的；本项目爬取各大漏洞情报来源（包括360，阿里，腾讯等漏洞来源）并及时推送到微信

## 脚本架构

```
│   README.md
|   start.sh        # 设置定时运行的脚本
├───threatpy
│   │   alithreaten.py
│   │   cert360threaten.py
│   │   config.py
│   │   dasthreaten.py
│   │   init_time.py
│   │   txthreaten.py
│   │   __init__.py
│   │
└───time
        alithreattime.txt
        cert360time.txt
        dasthreattime.txt
        txthreattime.txt
```
